### 手机端的新闻客户端

使用 __Kotlin__ 语言编写的 __Android__ 新闻客户端，
目前数据主要来源 __网易新闻__ 跟 __知乎日报__ 。

本项目主要用于学习，如数据涉及到版权，请联系作者。
